Setup:

1. Clone the repo

```bash
git clone git@github.com:murtylab/comp-neuro.git
```

2. Run the setup command:

```bash
python setup.py develop
```

3. Install some dependencies:

```bash
pip install torch numpy matplotlib scikit-learn
```
